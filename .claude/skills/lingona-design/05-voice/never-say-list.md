# Never-say list — hard-banned phrases

Catalog of phrases Lingona NEVER ships. If draft copy contains any phrase from this list → revise before commit.

This file = guardrail before code review. Search this file first when reviewing copy.

## 1. Corporate translate kill-list

Vietnamese phrases that translate corporate English directly. Sterile, distant, fake-formal. Lingona = peer voice mềm, NOT corporate.

| ❌ Banned | ✅ Replace with |
|----------|----------------|
| "Vui lòng {action}" | "{action} nhé" / direct verb |
| "Quý khách" | "bạn" |
| "Quý vị" | "bạn" |
| "Xin {action}" | "{action} nhé" / direct verb |
| "Hãy {action}" | "{action} đi" / "{action} nhé" |
| "Cần phải" | "luyện thêm" / specific verb |
| "Cảm ơn quý khách đã sử dụng" | "Cảm ơn bạn 🐙" / KHÔNG over-thank |
| "Trân trọng" | (skip — formality unnecessary) |
| "Kính chào" | "Chào" |
| "Liên hệ với chúng tôi" | "Nhắn cho mình qua {email}" |
| "Chúng tôi" (when refers to Lingona team) | "mình" / "Lingona" |
| "Hệ thống của chúng tôi" | "Lingona" / "mình" |
| "Người dùng" (when refers to actual user) | "bạn" |
| "Đăng ký ngay hôm nay" | "Đăng ký nhé" |
| "Bắt đầu ngay" | "Bắt đầu luyện" / "Bắt đầu đi" |
| "Đã xảy ra lỗi" | "Có lỗi rồi — thử lại nhé" |
| "Vui lòng thử lại" | "Thử lại nhé" |
| "Chúc mừng bạn" | "Vững rồi đấy" / specific praise |
| "Bạn đã hoàn thành" | "Xong rồi" / "Đã nộp bài" |
| "Bạn có muốn..." | "Bạn muốn..." (drop "có") |
| "Vui lòng đợi trong giây lát" | "Đợi tí nhé" / "Đang xử lý..." |

## 2. Cheerleading kill-list

Over-praising, fake encouragement. Phá credibility. User biết khi nào đang được nịnh.

| ❌ Banned | Why anti |
|----------|----------|
| "Bạn thật tuyệt vời!" | Over-praise, vague |
| "Bạn làm rất xuất sắc!" | Generic, fake |
| "Wow!" / "WOW!" | Cheerleader register |
| "Tuyệt!" / "Quá đỉnh!" | Vague intensity |
| "Cố lên bạn ơi!" | Caricature peer + ơi particle |
| "Bạn là nhất!" | Top-ranking unfounded |
| "Không có gì là không thể!" | Motivational poster cliché |
| "Bạn có thể làm được!" | Fake hype |
| "Hãy tin vào bản thân!" | Self-help cliché |
| "Mỗi ngày là một cơ hội..." | Inspirational quote register |
| "{X}+ year experience..." | Resume bullet cliché |

✅ Replace with **specific** praise:
- "Cohesion vững" (specific skill)
- "Top form đoạn này" (specific moment)
- "+0.5 so với lần trước" (factual progress)
- "Vững rồi đấy" (earned brief)

## 5. Saigon-dialect particle list

Vietnamese has regional dialects. Lingona uses neutral peer voice — KHÔNG bias regional.

| ❌ Banned particle | Region | Why anti |
|--------------------|--------|----------|
| **ơi** | Saigon | Caricature ("Bạn ơi!") |
| **nha** | Saigon | Too casual |
| **nè** | Saigon | Too casual |
| **đó** (post-imperative) | Saigon | "Vào đây đó" — over-casual |
| **vậy** (post-question) | Saigon | "Bạn muốn gì vậy?" — informal |
| **trời** / **trời ơi** | Universal slang | Non-professional |
| **má** / **má ơi** | Saigon | Slang |
| **ác** / **ác liệt** | Saigon slang | "Ác đấy!" — caricature |

KHÔNG dùng dialect markers. Use neutral peer particles: **nhé** / **đi** / **cùng**.

Hà Nội dialect cũng KHÔNG dùng:
- "ạ" (formal, anh/chị position): "Em chào anh ạ"
- "vâng" (over-formal yes)
- "thưa" (formal address)

## 11. Symbol / typography ban-list

| ❌ Banned | Why anti |
|----------|----------|
| `..` (2 dots, not ellipsis) | Typo |
| `...` (3 ASCII dots) — for ellipsis | Use `…` Unicode (U+2026) — exception: 3 dots OK in loading "Đang tải..." (signal animation) |
| `--` (2 hyphens) | Use `—` em-dash |
| `'` (straight apostrophe) | Use `'` curly apostrophe |
| `"` (straight quotes) | Use `"..."` curly quotes Vietnamese |
| `*` for emphasis (in user-facing copy) | Use `<strong>` HTML or markdown bold |
| `>` for blockquote (markdown literal in UI) | Use proper component |

NOTE: Markdown `**bold**` and `*italic*` ARE allowed in markdown skill files for documentation. Ban applies to user-facing UI copy.
