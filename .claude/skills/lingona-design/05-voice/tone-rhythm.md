# Tone rhythm — em-dash + peer particles + breathing

How Lingona's sentences move. Em-dash as breathing, peer particles as warmth, short clauses as direct.

## Em-dash signature — `—`

Em-dash (U+2014) is Lingona's signature punctuation. Used as **breathing pause** between clauses. Recognizable from 100ft per `00-manifesto/recognizable-from-100ft.md#4`.

NOT hyphen `-` (single character). NOT en-dash `–`. Em-dash specifically: `—` (long dash, ALT+0151 Windows / Option+Shift+Minus Mac).

### When em-dash

```
1. Pause for emphasis between thought + actionable
   ✅ "Đang gần Band 7 — luyện thêm linking words"
   ✅ "Cohesion vững — vocab cần đa dạng hơn"

2. Two-clause peer voice (state + suggest)
   ✅ "Phần này khó — luyện 3-5 bài rồi quay lại"
   ✅ "Streak 7 ngày — vững"

3. Insert mascot voice mid-sentence
   ✅ "Lintopus bảo — đoạn 2 cần liên kết hơn"

4. Skill summary one-liner
   ✅ "Reading Practice — Cambridge 14, Test 2"
   ✅ "Writing Task 2 — Opinion essay"
   ✅ "Speaking Part 2 — Long turn"
```

### When NOT em-dash

```
❌ Comma replacement (em-dash for everything = pretentious)
❌ Numerical range — use en-dash for ranges (Band 5.0–5.5 NOT 5.0—5.5)
❌ Sentence joiner without pause meaning
❌ Em-dash inside command/imperative ("Submit — bài làm")
```

```
✅ "Đang gần Band 7 — luyện thêm linking words"
❌ "Đang gần Band 7 — và — bạn nên — luyện thêm" (too many)
✅ "Band 5.0–5.5"  (en-dash for range)
```

### Em-dash spacing

Lingona uses **space around em-dash**:

```
✅ "Cohesion vững — vocab cần đa dạng hơn"  (space–em-dash–space)
❌ "Cohesion vững—vocab cần đa dạng hơn"  (no space, English typography)
```

Why space: Vietnamese typography traditionally spaces dashes. Looks cleaner.

## Sentence length rhythm

Mix short + medium clauses. Avoid uniformly long.

| Sentence type | Word count | Use |
|--------------|------------|-----|
| **Punchy** | 2-5 words | Status, label, dramatic moment |
| **Standard** | 5-12 words | Body copy, microcopy |
| **Two-clause em-dash** | 8-18 words | Peer voice signature |
| **Long** | 18-30 words | Detailed feedback (rare) |
| **Run-on** | 30+ words | ❌ Never — break apart |

Examples:

```
✅ Punchy:        "Vững rồi đấy"
✅ Punchy:        "Streak 7 ngày"
✅ Punchy:        "Còn 10 phút"

✅ Standard:      "Hôm nay luyện gì nhé?"
✅ Standard:      "Mạng yếu — đợi tí rồi thử lại"

✅ Em-dash:       "Đang gần Band 7 — luyện thêm linking words"
✅ Em-dash:       "Cohesion vững — vocab cần đa dạng hơn ở đoạn 2"

✅ Long (rare):   "Coherence ổn vì paragraphs có topic sentence rõ — nhưng vocab range hạn chế ở đoạn 2 và 3"

❌ Run-on:        "Bạn nên cố gắng cải thiện kỹ năng viết của mình bằng cách luyện tập thêm các bài tập về linking words và topic sentences để có thể đạt được band điểm cao hơn trong tương lai"
```

Rhythm cycle: punchy → standard → em-dash → punchy → ... User reads naturally without fatigue.

## Peer particles

Vietnamese has soft particles to soften imperative without losing direct. Lingona uses 4 main:

| Particle | When |
|----------|------|
| **nhé** | Soft suggestion, gentle imperative |
| **đi** | Direct invitation, casual |
| **cùng** | Inclusive ("with me / with us") |
| **với mình** | Lintopus mascot voice marker |

### nhé (soft suggestion)

```
✅ "Hôm nay luyện gì nhé?"
✅ "Lưu lại nhé"
✅ "Quay lại sau nhé"
✅ "Mình đợi bạn nhé"

❌ "Hôm nay luyện gì?" (too direct, demand register)
❌ "Hãy luyện hôm nay nhé!" (lecturing + force "hãy")
```

`nhé` adds warm tone without losing direct. Use for suggestion, gentle close.

### đi (direct invitation)

```
✅ "Bắt đầu đi"
✅ "Vào trận đi"
✅ "Cùng luyện đi"

❌ "Bạn vui lòng bắt đầu" (corporate)
❌ "Hãy đi nào!" (over-energetic)
```

`đi` casual + active. Use for action invitation, KHÔNG over-use (becomes pushy).

### cùng (inclusive)

```
✅ "Cùng luyện IELTS với mình"
✅ "Cùng vào trận nào"
✅ "Hôm nay cùng nhau ôn"

❌ "Mình và bạn luyện chung" (over-formal)
```

`cùng` includes user + Lintopus → companion register. Use sparingly, signature moments.

### với mình (Lintopus signature)

```
✅ "Cùng luyện IELTS với mình" (Landing hero)
✅ "Đi với mình tới Band 7" (motivational)
✅ "Practice với mình hằng ngày"

❌ "Đi với tôi" (too formal)
❌ "Cùng AI" (sterile)
```

`với mình` marker = Lintopus voice présence. Signature.

### Particles to AVOID

| Particle | Why anti |
|----------|----------|
| **ơi** | Too casual ("Bạn ơi!") — caricature |
| **nha** | Saigon dialect — bias regional |
| **nè** | Saigon dialect |
| **đấy** | OK in feedback ("vững rồi đấy") but careful overuse |
| **nào** | OK ("đi nào") but careful drama register |
| **được không** | Over-polite ("Bạn ấn nút được không?") — corporate |
| **xin** | Over-formal ("Xin mời bạn...") — corporate |

`đấy` OK but careful — quá nhiều thành annoying. Use 1x per page max.

```
✅ "Vững rồi đấy" (1x — moment)
✅ "Top form đấy" (1x — moment)
❌ "Hôm nay luyện đấy. Streak vững đấy. Lintopus tự hào đấy." (3x in row = annoying)
```

## Imperative softening

Vietnamese imperative without softener = harsh. Lingona always soften:

| Hard | Soft |
|------|------|
| "Nhấn nút" | "Nhấn nút nhé" / "Bấm vào đây" |
| "Đăng nhập" | "Đăng nhập nhé" / "Đăng nhập để tiếp tục" |
| "Chọn band" | "Bạn chọn band nào?" / "Chọn band của bạn" |
| "Submit" | "Nộp bài đi" / "Nộp bài ⚔️" |
| "Click here" | "Bấm vào đây" / "Vào đây nhé" |

Soft variant: particle (nhé / đi) OR question form OR pronoun "bạn".

## Question form

Convert imperative to question = inviting:

```
✅ "Hôm nay luyện gì nhé?"
✅ "Bạn muốn thi thật hay luyện?"
✅ "Bắt đầu nhé?"

❌ "Hãy chọn chế độ luyện" (lecturing)
❌ "Vui lòng chọn chế độ" (corporate)
```

Question form = peer (asking permission, not commanding).

## Avoid run-on / lecture

Lecture register: long sentence + multiple "để" clauses + passive verbs.

```
❌ Lecture:
"Bạn nên cố gắng cải thiện kỹ năng viết của mình bằng cách luyện tập thêm các bài tập về linking words để có thể đạt được band điểm cao hơn trong tương lai gần."

✅ Peer voice:
"Vocab vững rồi — luyện thêm linking words để lên Band 7."
```

Difference:
- 1 thought, not 3
- Em-dash breath
- Direct verb ("luyện thêm") not modal ("nên cố gắng")
- Specific (linking words) not vague (kỹ năng viết)

## Tone gradient per surface

Voice tone slides slightly per surface:

| Surface | Tone | Example |
|---------|------|---------|
| Landing hero | Warm + confident | "Cùng luyện IELTS với mình" |
| Onboarding | Curious + attentive | "Bạn đang ở band nào?" |
| Dashboard | Practical + low-key | "Hôm nay luyện gì nhé?" |
| In-task chrome | Minimal + factual | "Còn 10 phút" |
| Mid-task hint | Quiet + helpful | "Gợi ý: dùng 'however' để contrast" |
| Result high band | Earned + brief | "Vững rồi đấy" |
| Result mid band | Specific + direct | "Cohesion vững — vocab cần đa dạng hơn" |
| Result low band | Companion + recovery | "Lintopus đợi bạn ở vòng tiếp theo" |
| Battle queue | Tense + brief | "Đang tìm đối thủ..." |
| Battle result | Drama register | "VICTORY — top form đấy" |
| Error state | Factual + recovery | "Lintopus đang sửa! 🐙" |
| Empty state | Inviting + clear | "Chưa có bài. Luyện bài đầu tiên nhé." |
| Legal | Formal + Vietnamese | "Chính sách hoàn tiền 7 ngày..." |
| Critical (delete confirm) | Sober + factual | "Hành động không thể hoàn tác" |

Same persona, different volume. Like classmate adjusting tone for context.

## Anti-rhythm patterns

❌ **Monotone declarative** — "Bạn đăng nhập. Bạn chọn skill. Bạn luyện." (robot)
❌ **Run-on lecture** — 30+ words trong 1 sentence
❌ **Comma overuse** — "Bạn vui lòng, hãy chọn skill, để có thể bắt đầu, luyện tập" (each comma = stutter)
❌ **Em-dash spam** — "Hôm nay — luyện — band — 7 — nhé" (broken)
❌ **Particle stacking** — "Bạn ơi, vào đây nhé nhỉ?" (too many)
❌ **Translate cadence** — "Bằng cách sử dụng Lingona, bạn có thể đạt được..." (English structure VN)

## Audit checklist

```
1. Em-dash signature present where peer voice? ✓
2. Em-dash with space around (— ) NOT no-space? ✓
3. Sentence rhythm mix (punchy + standard + em-dash)? ✓
4. NO run-on (>30 words)? ✓
5. Particles soften imperative (nhé/đi/cùng) per surface? ✓
6. NO Saigon dialect particles (ơi/nha/nè) — too casual? ✓
7. NO corporate particles (xin/vui lòng/quý) — too formal? ✓
8. Question form for invitations (NOT lecture imperative)? ✓
9. Specific verb (luyện thêm) NOT modal (nên cố gắng)? ✓
10. Tone gradient appropriate per surface? ✓
```
