# Persona — bạn cùng lớp

Lingona's voice persona = **"bạn cùng lớp"** (peer classmate). Per `00-manifesto/personality.md` lock.

This file expands persona scope: register details, when to break peer voice, how to embody persona without caricature.

## Persona definition

**Bạn cùng lớp** = classmate at same IELTS level as user, slightly ahead in journey, helping out without showing off.

NOT:
- Tutor (anh/chị position implies authority)
- Senior student (anh khóa trên / chị khóa trên — implies hierarchy)
- AI assistant (Bot voice — corporate translate)
- Coach (English fitness register — sterile)
- Friend (too informal, breaks IELTS-prep seriousness)

**Bạn cùng lớp** = peer + practical + warm + competent + zero ego.

## Pronoun lock — mình / bạn

**Mandatory across all Lingona surface** (except IELTS-authentic mode where English UI takes over):

| ✅ Use | ❌ Banned |
|--------|-----------|
| **mình** (self-reference) | tao, anh, chị, em (in default brand mode) |
| **bạn** (user-reference) | mày, em, anh, chị, quý khách, quý vị |
| Implicit no-pronoun | Bạn cùng các bạn (over-formal Vietnamese plural) |

```
✅ "Mình thấy bạn đang gần Band 7"
✅ "Cùng luyện với mình nhé"
✅ "Bạn muốn thử Speaking trước không?"

❌ "Tao thấy mày đang gần Band 7" (too casual, Saigon street)
❌ "Em thấy anh đang gần Band 7" (gender + age assumption)
❌ "Quý khách muốn thử Speaking?" (corporate, distance)
❌ "AI thấy user đang gần Band 7" (sterile, English-mode bleed)
```

**Edge case**: If addressing system-wide notification (no specific user yet, e.g., "Welcome screen"):
- Use **bạn** generically still works
- KHÔNG fall back to passive "Người dùng được thông báo..." (corporate)

```
✅ "Chào bạn! Lingona giúp bạn luyện IELTS."
❌ "Chào người dùng! Hệ thống giúp luyện IELTS." (sterile)
```

## Persona embodiment per surface

### Landing page

Persona **not yet** revealing — Lintopus visual + headline establish identity. Voice subtle:

```
✅ "Cùng luyện IELTS với mình"
✅ "Mình giúp bạn từ Band 5.0 đến 7.5+"
✅ "Phản hồi thực tế. Không cảm tính."
```

Avoid:
- Over-friendly "Hey bạn ơi!" (caricature)
- Corporate distance "Lingona — Nền tảng học IELTS hàng đầu Việt Nam"
- AI-template "Discover the power of AI-driven IELTS prep"

### Onboarding

Persona warming up. Asks questions like classmate:

```
✅ "Bạn đang ở band nào?"
✅ "Mục tiêu của bạn là gì?"
✅ "Mỗi ngày bạn có khoảng bao lâu?"

❌ "Vui lòng chọn trình độ hiện tại" (corporate)
❌ "Please select your current level" (English bleed)
```

### Dashboard greeting

Daily check-in tone, no gushing:

```
✅ "Chào {username} 🐙"
✅ "Hôm nay luyện gì nhé?"
✅ "Streak 7 ngày — vững"

❌ "Chào mừng quay trở lại!" (corporate)
❌ "Welcome back, {user}!" (English)
❌ "Lingona rất vui được gặp lại bạn!" (formal+gushing)
```

### Mid-task encouragement

Sparse, factual peer comments. NOT cheer constant:

```
✅ "Đã viết 150 / 250 từ — tiếp tục"
✅ "Còn 10 phút"
✅ "Phần này khó. Cứ thử."

❌ "Bạn đang làm rất tốt! Cố lên!" (cheerleading constant)
❌ "Keep going! You can do it!" (English)
❌ "Một nửa rồi đó! Cố thêm chút nha!" (over-friendly)
```

### Result page

Soul §1 stand next to result. Specific + actionable + peer:

**High band (7.0+)**:
```
✅ "Vững rồi đấy"
✅ "Top form đoạn này"
✅ "Kỹ thuật ổn — giữ rhythm này"

❌ "WOW! Bạn thật tuyệt vời!" (over-celebrate)
❌ "Excellent work! Amazing!" (English fake-cheer)
❌ "Bạn đã làm rất xuất sắc!" (corporate praise)
```

**Mid band (5.5-6.5)**:
```
✅ "Đang gần Band 7"
✅ "Còn vài chỗ luyện thêm"
✅ "Cohesion vững — vocab cần đa dạng hơn"

❌ "Cố gắng thêm chút nữa!" (vague, blame-implied)
❌ "Need improvement" (cold + English)
❌ "Bạn cần phải cải thiện..." (lecturing)
```

**Low band (<5.0)**:
```
✅ "Lintopus đợi bạn ở vòng tiếp theo"
✅ "Phần này khó. Luyện thêm 3-5 bài rồi quay lại"
✅ "Foundation cần củng cố — bắt đầu từ Practice mode"

❌ "Đừng buồn, bạn sẽ làm được!" (fake comfort)
❌ "Don't worry, you'll get there!" (English fake)
❌ "Kết quả chưa tốt. Hãy cố gắng hơn." (judgment)
```

### Error states

Factual, non-blaming:

```
✅ "Không kết nối được. Thử lại nhé."
✅ "Lintopus đang sửa! 🐙"
✅ "Mạng yếu — đợi tí rồi thử lại"

❌ "Đã xảy ra lỗi. Vui lòng thử lại." (corporate)
❌ "Something went wrong" (English)
❌ "Lỗi nghiêm trọng đã xảy ra! Liên hệ quản trị viên!" (drama)
```

### Empty states

Encouraging next action, not apologizing for emptiness:

```
✅ "Chưa có bài nào. Luyện bài đầu tiên nhé."
✅ "Friends danh sách trống — mời bạn qua link 🐙"
✅ "Lịch sử trống — kết quả đầu tiên sẽ hiện ở đây"

❌ "Không có dữ liệu" (sterile)
❌ "No data found" (English)
❌ "Rất tiếc, bạn chưa có hoạt động nào!" (over-apologetic)
```

### Battle drama (exception)

Battle = drama register, persona shifts slightly more energetic. See `05-voice/battle-drama.md`.

```
✅ "VICTORY — top form đấy"
✅ "DEFEAT — Lintopus đợi bạn ở vòng tiếp theo"
✅ "Tìm đối thủ..."

❌ "GG WP" (gaming slang — out of register)
❌ "ez win" (toxic gaming)
❌ "Better luck next time!" (English)
```

## When to break peer voice

Peer voice mềm OK for 90% of surface. Exception cases:

### 1. IELTS-authentic mode (Full Test, Battle Ranked match)

Voice swaps to **English neutral exam**. KHÔNG peer voice during exam.

```
Brand mode:        "Nộp bài ⚔️"
IELTS-authentic:   "Submit"

Brand mode:        "Bạn muốn thi thật?"
IELTS-authentic:   "Begin Full Test"
```

See `04-modes/ielts-authentic.md`.

### 2. Legal / policy text

Privacy policy, Terms of Service, Refund policy — neutral formal but Vietnamese:

```
✅ "Chính sách hoàn tiền: 7 ngày kể từ thanh toán..."
✅ "Bằng việc sử dụng dịch vụ, bạn đồng ý..."

❌ "Mình áp dụng chính sách hoàn tiền 7 ngày nhé!" (peer in legal = unprofessional)
```

Per Wave 6 launch audit — legal pages use formal but Vietnamese-first.

### 3. Critical safety message

Account deletion confirm, financial action, data loss warning:

```
✅ "Hành động không thể hoàn tác"
✅ "Sau 7 ngày, dữ liệu sẽ bị xóa vĩnh viễn"
✅ "Gõ XÓA để xác nhận"

❌ "Hơi tiếc đó nha — sure muốn xóa?" (peer voice = un-serious in legal-critical)
```

Critical context = formal Vietnamese, KHÔNG peer.

### 4. Numbered list / table content

Data display in table doesn't need persona injection:

```
✅ "Band 7.0", "Coherence: 7.5", "Vocabulary: 7.0"
✅ "07/12/2026 · Reading Cambridge 14"

❌ "Band 7.0 vững rồi đấy" trong cell table (over-narrate cell)
```

Persona = chrome voice (microcopy, headlines, action). Data = neutral.

## Anti-personas

❌ **Cheerleader Lintopus** — "Wow! You're doing amazing! Keep it up!"  → over-cheerful, fake
❌ **Strict tutor Lintopus** — "You need to improve. Practice more." → judgmental, anh/chị position
❌ **Corporate Lintopus** — "Vui lòng cải thiện kỹ năng" → distance, sterile
❌ **Gen Z friend Lintopus** — "Bạn ơi xịn xò ghê!" → caricature, breaks credibility
❌ **English-mode Lintopus** — "Great job!" → bleed from English context
❌ **Self-deprecating Lintopus** — "Mình xin lỗi vì..." → fake humility
❌ **Over-explaining Lintopus** — "Mình đã phân tích kỹ lưỡng..." → robotic
❌ **Passive-aggressive Lintopus** — "Bạn đã không luyện 3 ngày rồi!" → Duolingo Owl pattern (anti per `00-manifesto/what-we-are-not.md`)

## Voice DNA (5 traits)

Distill peer voice across surface:

1. **Specific** — "Cohesion vững — vocab cần đa dạng hơn ở đoạn 2" (NOT "Try harder")
2. **Direct** — "Còn 10 phút" (NOT "Bạn còn 10 phút thôi đấy nhé!")
3. **Em-dash rhythm** — "Đang gần Band 7 — luyện thêm linking words" (breathing pause)
4. **Vietnamese-first** — full Vietnamese always, English only Battle drama exception
5. **Zero ego** — "Mình giúp bạn" (NOT "AI của Lingona giúp bạn")

5 traits applied = consistent persona across surface.

## Quick voice check (3 questions)

Before shipping any copy, ask:

```
1. Pronoun = mình/bạn? (KHÔNG anh/chị/quý)
2. Specific or vague? (specific = ship, vague = revise)
3. Em-dash rhythm or run-on? (em-dash = ship)

Pass 3/3 = peer voice OK
Fail any = revise
```

## Audit checklist

```
1. Pronoun mình/bạn applied? ✓
2. NO anh/chị/em/quý/tao/mày? ✓
3. NO English copy in brand surface? ✓ (except Battle drama)
4. Specific feedback (skill name + action) NOT vague? ✓
5. Em-dash rhythm where peer voice? ✓
6. Anti-cheerleading (no constant praise)? ✓
7. Anti-corporate (no "vui lòng" / "quý khách")? ✓
8. Mode-aware: IELTS-authentic = English? ✓
9. Legal/critical = formal VN (not peer)? ✓
10. Lintopus in voice scope = visual + bubble text only (no audio)? ✓
```
